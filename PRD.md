# PRD：大模型开发个人作品集网站

## 1. 背景与动机

正在系统学习大模型开发，方向涵盖 RAG、Agent、向量数据库等，目标是获得大模型开发实习岗位。

目前学习内容分散，缺乏统一的展示平台。本站旨在构建一个**面向面试官的个人技术作品集**，用真实项目和技术笔记证明技术理解深度，辅助求职。

---

## 2. 目标用户

- **主要用户：** 自己（学习追踪 + 自我激励）
- **次要用户：** 潜在招聘者、同学、技术社区（展示用途）

---

## 3. 核心目标

1. 将所有大模型相关项目统一收录，可视化进度
2. 整合优质学习资料（教程、论文、工具、课程）
3. 提供清晰的"大模型学习路径"导航
4. 支持快速扩展（新项目 / 新模块随时加入）

---

## 4. 功能模块

### 4.1 首页 Hero 区
- 简短的个人介绍（"我在系统学习大模型开发"）

- 完成状态（规划中 / 进行中 / 已完成）
- 链接（GitHub / 演示地址）

| （待补充）多模态项目 | 多模态 LLM | 规划中 |

### 4.3 学习路径区（Learning Path）
用时间线或阶段图呈现大模型学习的推荐路径：

```
阶段 1：基础层
  - Transformer 架构理解
  - API 调用（OpenAI / Claude / Gemini）
  - Prompt Engineering

阶段 2：应用层
  - RAG（检索增强生成）
  - LangChain / LlamaIndex

阶段 3：深化层
```

每个阶段标记当前所处位置。

## 5. 技术选型建议

| 层面 | 选项 | 推荐理由 |
|------|------|----------|
| 框架 | Next.js 15 (App Router) | 生态成熟，SSG 适合展示站 |
| 样式 | Tailwind CSS + shadcn/ui | 快速出效果，组件丰富 |
| 内容管理 | MDX 文件 + 本地 JSON | 简单，无需后端，易维护 |
| 部署 | Vercel | 与 Next.js 无缝集成，免费额度够用 |
| 动效 | Framer Motion | 进度条、卡片动画 |

> 如果希望更快 vibe coding，也可以考虑用 **Astro + Tailwind** 纯静态方案，构建速度更快。

## 6. 页面结构（信息架构）

```
/                     首页（Hero + 项目速览 + 学习路径预览）
/projects             项目列表（全部项目卡片）
/projects/[slug]      项目详情页
/learning-path        完整学习路径（时间线视图）
/resources            资料库（分类筛选）
/about                关于我
```

---

## 7. 设计风格参考

- **整体基调：** 深色主题（dark mode first）+ 技术感
- **配色参考：** 深灰背景 + 青色/紫色高亮（参考 Vercel 官网、Linear 官网）
- **字体：** Geist Sans（Next.js 默认）或 Inter
- **视觉元素：** 渐变边框卡片、代码字体点缀、进度条表示学习状态

---

## 8. MVP 范围（第一版交付目标）

第一版只做以下内容，快速上线：

- [x] 首页 Hero 区
- [x] 项目展示区（静态数据，3-5 个项目）
- [x] 学习路径区（静态时间线）
- [x] 资料库区（基础分类，10-20 条）
- [x] 关于页

**暂不做：**
- 博客 / 笔记系统（后期加）
- 搜索功能
- 评论系统
- 后台 CMS
